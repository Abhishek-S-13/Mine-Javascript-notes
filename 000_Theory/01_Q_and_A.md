## ✅ Basic JavaScript Theory Questions:
```Javascript
1. What is JavaScript?
-> A scripting language used to create and control dynamic website content.

2. Difference between JavaScript and Java?
-> They are unrelated: Java is a full programming language; JavaScript is mainly for web interaction.

3. What are variables in JavaScript?
-> Containers for storing data values (e.g., var, let, const).

4. What are data types in JavaScript?
-> Examples: String, Number, Boolean, Object, Null, Undefined, Symbol, BigInt.

5. Difference between let, const, and var?
-> var is function-scoped; let and const are block-scoped. const cannot be reassigned.

6. What is hoisting in JavaScript?
-> JavaScript moves declarations to the top of the scope before execution.

7. What are functions?
-> Blocks of code designed to perform a particular task.

8. What is the difference between == and ===?
-> == checks value (with type conversion); === checks value and type.

9. What are objects and arrays in JavaScript?
-> Objects store data in key-value pairs; arrays store ordered lists of values.

10. What is the DOM?
-> The Document Object Model — a tree structure representing HTML elements that JavaScript can interact with.
```
